# React Feature Explorer

This project have the main purpose to expore freely new features and libs
At this momment yoou can find those little sub-project:
  - Autocomplete : A simply exploration of how to make an autompletion input component
  - Todolist : A fully feature Drag and Drop to list using the localstorage as fake DB

## How to install

```shell
npm install
```

## File Structure
This project use the feature based file structure
```
src
└───features
│   └─── shared
│       └─── containers
│       └─── components
│       │    └───   ... 
│       └─── ...
│   └─── todolist
│        └─── containers
│        └─── components
README.txt
```
